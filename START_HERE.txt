1️⃣  OPEN: VSCode_LOCAL_SETUP.md (Complete step-by-step guide)

═══════════════════════════════════════════════════════════════

🎯 FASTEST WAY TO GET STARTED:

  Mac/Linux Users:
  ▶ ./run_locally.sh

  Windows Users:
  ▶ run_locally.bat

  Then open TWO terminals:
  Terminal 1: cd backend && python3 server.py
  Terminal 2: cd frontend && yarn start

  Open browser: http://localhost:3000
